# Wireless Assistive Hearing Aid

## Overview
The Wireless Assistive Communication Device is designed to help individuals with hearing and speech impairments communicate effectively. The system uses embedded technology and wireless communication to convert user inputs into text and speech, enabling seamless interaction in real-time.

## Objectives
- Facilitate communication for deaf and mute individuals.
- Convert predefined inputs into text and speech.
- Enable wireless transmission of messages.
- Develop a portable and user-friendly assistive device.
- Improve accessibility and communication efficiency.

## Technologies Used
- Arduino / ESP32
- Embedded C
- Bluetooth / Wi-Fi Module
- LCD Display
- Speaker / Buzzer
- Arduino IDE

## Working Principle
1. User selects a predefined message using buttons or sensors.
2. The microcontroller processes the input.
3. The message is displayed on the LCD screen.
4. Text-to-speech functionality generates audio output.
5. The message is transmitted wirelessly to the receiver.
6. The receiver displays the message for effective communication.

## Features
- Real-time communication support
- Wireless message transmission
- Text-to-speech conversion
- Portable and lightweight design
- Easy-to-use interface
- Cost-effective assistive solution

## Applications
- Healthcare and hospitals
- Educational institutions
- Public service centers
- Home assistance
- Assistive technology solutions

## Future Enhancements
- Gesture recognition using Machine Learning
- Mobile application integration

## Results
The developed prototype successfully enabled wireless communication by transmitting predefined messages and converting them into text and speech outputs. The system demonstrated its potential as an affordable assistive technology solution for individuals with hearing and speech impairments.
